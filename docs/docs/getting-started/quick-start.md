# Quick Start

Get up and running with github-actions-workflow-ts in minutes.

## Create Your First Workflow

Create a new file with the `.wac.ts` extension anywhere in your project. For example, `workflows/ci.wac.ts`:

```typescript
// workflows/ci.wac.ts
import { Workflow, NormalJob, Step } from '@github-actions-workflow-ts/lib'

// Define a step
const checkoutStep = new Step({
  name: 'Checkout',
  uses: 'actions/checkout@v4',
})

// Define a job with steps
const testJob = new NormalJob('Test', {
  'runs-on': 'ubuntu-latest',
  'timeout-minutes': 10,
})

testJob.addStep(checkoutStep)
testJob.addStep(
  new Step({
    name: 'Run Tests',
    run: 'npm test',
  })
)

// Define and export the workflow
// IMPORTANT: Every Workflow instance MUST be exported
export const ciWorkflow = new Workflow('ci', {
  name: 'CI',
  on: {
    push: {
      branches: ['main'],
    },
    pull_request: {
      branches: ['main'],
    },
  },
})

ciWorkflow.addJob(testJob)
```

## Generate the YAML

Run the CLI to generate the workflow YAML file:

```bash
npx gwf build
# or
npx generate-workflow-files build
```

This creates `.github/workflows/ci.yml`:

```yaml
# ----DO-NOT-MODIFY-THIS-FILE----
# This file was automatically generated by github-actions-workflow-ts.
# Instead, modify `workflows/ci.wac.ts`
# ----DO-NOT-MODIFY-THIS-FILE----

name: CI
on:
  push:
    branches:
      - main
  pull_request:
    branches:
      - main
jobs:
  Test:
    runs-on: ubuntu-latest
    timeout-minutes: 10
    steps:
      - name: Checkout
        uses: actions/checkout@v4
      - name: Run Tests
        run: npm test
```
